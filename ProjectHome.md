A RepRap Emulator that is easily extensible and has a nice interface.

**Features**
  * Interface
  * Multiple Speeds
  * Multiple Firmwares
    * Full Implementation
    * Generation 5
    * RepMan (Mostly Implemented)
  * PostScript Output
  * Nine Speeds
  * Multiple Inputs
    * GCode File
    * Serial Port
    * UNIX Socket (VirtualBox Serial)
